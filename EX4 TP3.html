DEFINITIONS

RSS
Un format XML servant à réunir des informations souvent mises à jour d'un site Internet.
Weblog
Contraction des termes "web" et "log", un weblog est un site souvent mis à jour, à l'image d'un journal.
HTTP
Hyper Text Transfert Protocol est un protocole qui définie la communication entre un client ( exemple : navigateur ) et un serveur sur le World Wide Web (www).
HTML.5
Hyper Text Markup Language version 5 est la dernière version en date du langage de développement web HTML.
